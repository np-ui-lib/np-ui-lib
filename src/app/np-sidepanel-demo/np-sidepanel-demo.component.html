How to set lazy loading for sidepanel?
Property Description Type Default
position Set position 'left' | 'right' | 'top' | 'bottom' string left
height Set height. Values should be like '100%', '100px', 200. If pass as number then it will considered as px. string | number
width Set width. Values should be like '100%', '100px', 200. If pass as number then it will considered as px. string | number
closeOnClickOutside Close panel on click of outside area. If backdrop is not set then close on outside click will not work. boolean true
hasBackDrop Show backdrop or not boolean true
backDropClass Set backdrop class string np-sidepanel-backdrop
styleClass Set extra class on component string
inputId Set id attribute string
Method Description Parameters
onOpen(data : any) Callback on open data passed from open api
onClose(data : any) Callback on close data passed from close api
Api Description Parameters
open(data : any) Open side panel Pass data to onOpen method
close(data : any) Close side panel Pass data to onClose method
