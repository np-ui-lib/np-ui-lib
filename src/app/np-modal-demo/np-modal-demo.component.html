Modal
Module to import
NpModalService

Inject NpModalService into constructor of component

 How to open modal?
open(content: string | TemplateRef<any> | Type<any>, config: NpModalConfig, data: any): NpModalRef;

Here open method has 3 parameters, and this method return the overlay refrence object of class NpModalRef

Parameter Description Type
content Pass content of modal
string | TemplateRef<any> | Type<any>
config Pass modal config
Which has properties like header, height, width, hasBackDrop, backDropClass, showCloseButton, closeOnClickOutside, styleClass, inputId.
NpModalConfig
data Pass data to modal any
How to subscribe onClose method of modal?

open method returns overlay refrence object, which has property onClose: Subject<any>

 How to access data and NpModalRef object in another component?

In another component, overlay refrence is injected using injector.

Here ref object contains data: any and close(data: any) method. Which will close overlay.

Pass response data in close(data: any) method's parameter, which will be accessible where onClose is subscribed.

string
TemplateRef<any>
