Server side basic data grid example

To add server side oprations in data grid, set isServerOperations to "true" in grid defination.

And also add method onLoadData(options:LoadOptions). This method will be invoked every time to fetch data. 

{{htmlText}}
{{gridDataSourceText}}

LoadOptions has below properties:

  • pageNumber
  • pageSize
  • sortColumns (This will contains array of all sort columns as {{ '{' }} column : string, sortDirection: string {{'}'}})
  • filterColumns (This will contains array of all filter columns as {{ '{' }} column : string, filterOperator: string, filterValue: string, dataType: string {{'}'}})
  • odataQuery (If isODataOperations is set to "true", then oDataQuery will be passed here.)
  • isAllPages (This will be used for server side export to CSV functionality, default value is "false")

Possible values for sortDirection are asc, desc.

Possible values for filterOperator are startswith, endswith, contains, gt, lt, ge, le, eq, ne.

Possible values for dataType are number, string, date, boolean.
