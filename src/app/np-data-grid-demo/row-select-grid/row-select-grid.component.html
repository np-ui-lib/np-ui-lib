Row Selction Example

To set single row selection, set singleRowSelectEnable to "true" in grid defination

To set multi row selection, set multiRowSelectEnable to "true" in grid defination

In multi row selection, Select All checkbox is given in table header to select/unselect all rows

onSelect and onDeselect events:

In both single/multi select, onSelect($event), onDeselect($event) methods can be bound

This methods will be run when any row will be selected or de-selected. In $event.data, affected row keys will be passed.

getSelectedRowKeys API:

Here getSelectedRowKeys() API is also available, which will return array of all selected rows keys

On reset of data grid, all selected keys will be removed

selectRowOnClick

To select row on click set selectRowOnClick to true in grid defination
